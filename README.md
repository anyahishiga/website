# 🍽️ H&S Eat - Restaurant Review Website

H&S Eat is a restaurant review website that helps users discover, explore, and review restaurants and local food destinations. The platform provides detailed restaurant information, ratings, user reviews, and an intuitive interface for food lovers.

---

## 📖 Overview

H&S Eat is designed to make finding great food easier by allowing users to:

- 🔍 Browse restaurants
- ⭐ View ratings and reviews
- 📝 Submit their own reviews
- 📍 Explore restaurant details
- ❤️ Discover recommended places

The project focuses on creating a clean, responsive, and user-friendly experience.

---

## ✨ Features

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/website.git
```

### Open the project

Simply open the project folder in Visual Studio Code.

### Run the website

If using **Live Server**:

1. Install the Live Server extension.
2. Right-click `html/index.html`
3. Select **Open with Live Server**.
